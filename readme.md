# Real-Time Network Intrusion Detection System

## Project Overview
This project implements a **Machine Learning based Network Intrusion Detection System (NIDS)** that detects malicious network traffic in real-time.

The system analyzes network traffic features and predicts whether the traffic is **Normal or an Intrusion Attack**.

## Dataset
We used the **NSL-KDD dataset**, which is a benchmark dataset for network intrusion detection research.

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Machine Learning

## Features
- Real-time intrusion prediction
- Interactive dashboard
- Cybersecurity themed UI
- Machine learning based attack detection

## Machine Learning Model
The model was trained using supervised machine learning techniques on the NSL-KDD dataset.

Algorithms used:
- Random Forest
- Logistic Regression
- Bagging Classifier

## Dashboard
The system includes a **Streamlit dashboard** that allows users to:

- Input network traffic features
- Detect intrusions
- View traffic analysis graphs
- Monitor system statistics
